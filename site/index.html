Tutorial

添加教程

  1. 取得 fakeid。登录微信公众平台，在图文编辑器的“超链接 / 选择其他公众号文章”中搜索目标公众号；打开浏览器开发者工具 Network，找到名称含 searchbiz 的请求，在响应中复制 fakeid。
  2. 不要填 gh_ 开头的 user_name。本服务需要 Base64 风格 fakeid，通常类似 MzA...==。
  3. 生成配置。在左侧填名称和 fakeid，加入草稿后复制完整 JSON。
  4. 提交到 GitHub。打开编辑页，全选替换 config/accounts.json 并提交。
  5. 让配置生效。等待次日 10:00，或手动运行 Daily WeChat job scan ↗。
为什么不能直接保存公众号？

GitHub Pages 是静态网页。把仓库写入令牌放进页面会让其他访客也能修改配置，因此最后一步仍需在已登录的 GitHub 编辑页提交。

自动筛选仅用于减少信息噪声，岗位要求请以公众号原文与招聘单位官网为准。
